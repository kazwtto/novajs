NOVAjs

Uma linguagem de script limpa e expressiva que transpila para JavaScript. Escreva menos código boilerplate, mantenha total compatibilidade com JS.

Nova é uma linguagem que compila diretamente para JavaScript. Ela mantém o ecossistema JS totalmente intacto, ao mesmo tempo que fornece uma sintaxe mais limpa: ponto e vírgula opcionais, operadores baseados em palavras, declarações de função mais curtas, retornos implícitos, interpolação rica de strings e APIs DOM + Canvas de primeira classe.

Instalação

Navegador (tag script)

<script src="caminho/para/nova-lang.js"></script>
<script type="text/novajs">
  print("Olá, mundo!")
</script>

Node.js (CLI)

npm install -g nova-lang 
# Executar um arquivo
nova script.nova

# Código inline
nova "print('Olá!')"

Node.js (API)

const Nova = require('nova-lang')

Nova.transpile(src)          // → string JS
await Nova.executeNode(src)   // transpilar + executar

Variáveis

Nova suporta let, const e var. var é normalizado para let. Um identificador em TUDO_MAIÚSCULO atribuído no nível da declaração se torna um const implícito. 

let x = 10
                const PI = 3.14
                TAMANHO_MAXIMO = 100       // → const TAMANHO_MAXIMO = 100

                // A palavra-chave 'is' funciona como atribuição
                let pontuacao is 0
                pontuacao is 42

Declaração implícita

Um identificador seguido por : NomeDoTipo no nível da declaração também cria um let automaticamente. 

contagem: number = 0   // → let contagem = 0

Sistema de Tipos

Tipos são opcionais e verificados em tempo de execução quando anotados. Erros de tipo geram mensagens descritivas. Usar any desabilita a verificação. 

let n: number  = 42
let s: string  = "oi"
let b: boolean = true
let a: array   = []
let o: object  = {}
let f: func    = () => {}
let v: any     = "qualquer coisa"

number num, int, float typeof === 'number'
string str, texto typeof === 'string'
boolean bool typeof === 'boolean'
object obj typeof === 'object'
array — Array.isArray()
function func, fn typeof === 'function'
null — === null
undefined — === undefined
any — sem verificação

Tipos de formato de objeto

let usuario: { nome: string, idade: number } = { nome: "Ana", idade: 30 }

// Campo opcional com ?

Strings

Nova suporta aspas simples, aspas duplas e template literals com crase. Interpolação de strings funciona dentro de qualquer estilo de aspas usando $var ou ${expr}. 

let nome = "mundo"

// Interpolação de variável
print("Olá, $nome!")

// Interpolação de expressão
print("2 + 2 = ${2 + 2}")

// Template literal (crase)
print(`Nome: ${nome.upper()}`)

// Template multi-linha
let html = `
  <div class="card">
    <h2>$nome</h2>
  </div>
`

Funções

Funções podem ser declaradas com func, fn ou function. A última expressão no corpo da função é retornada implicitamente. Funções de uma linha omitem as chaves completamente. 

// Uma linha (corpo de expressão, retorno implícito)
func dobrar(x) x * 2

// Corpo com blocos (última expressão retornada)
fn clamp(v, min, max) {
  v < min ? min : v > max ? max : v
}

// Parâmetros padrão
func saudacao(nome = "mundo") "Olá, $nome!"

// Parâmetros rest
func somar(...nums) nums.reduce((a, b) => a + b, 0)

// Arrow functions
const quadrado = x => x * x
const adicionar = (a, b) => a + b

// Async / generator
async func buscarDados(url) {
  let res = await fetch(url)
  await res.json()
}

// Corpo literal de objeto (retorno implícito)
func criarUsuario(nome, idade) { nome: nome, idade: idade }

Suporta um sistema de tipos flexível e expressivo com sintaxe concisa de função, permitindo anotações opcionais, tipos anuláveis e uniões, e retornos implícitos para um código mais limpo. 

fn dobrar(x: number): number x * 2            // uma linha
fn adicionar(a, b): number { a + b }                // bloco, retorno implícito
fn encontrar(id): string? { id == 1 ? "x" : null } // anulável
fn analisar(s): number|null { s.number() }       // união
fn registrar(msg): void { print(msg) }              // void

Sintaxe de Método com Dois Pontos

Nova fornece uma sintaxe alternativa para chamada de métodos usando :. Funciona em qualquer objeto, classe, palavra-chave, this ou acesso encadeado.

objeto: metodo() → objeto.metodo()

arr: push(42)              // arr.push(42)
arr: join(", ")            // arr.join(", ")
str: toUpperCase()         // str.toUpperCase()
Math: max(1, 2, 3)         // Math.max(1, 2, 3)
new Date(): toISOString()  // new Date().toISOString()

Encadeamento também funciona:

arr: filter(x => x > 2): map(x => x * 10): join(", ")
// arr.filter(x => x > 2).map(x => x * 10).join(", ")

objeto(arg: metodo) → objeto.metodo(arg)

lista(42: push)             // lista.push(42)
lista("item": push)         // lista.push("item")

objeto(a: m1, b: m2) → múltiplas chamadas

pilha("a": push, "b": push, "c": push)
// pilha.push("a"); pilha.push("b"); pilha.push("c")

Funciona com this e acesso a propriedades:

this.partes(item: push)     // this.partes.push(item)
c: incrementar()             // c.incrementar()
dom.get("#app"): esconder()    // dom.get("#app").esconder()

Fluxo de Controle

if / else

Parênteses ao redor da condição são opcionais.

if x > 0 {
  print("positivo")
} else if x < 0 {
  print("negativo")
} else {
  print("zero")
}

// Ternário
let rotulo = x > 0 ? "pos" : "não-pos"

switch

switch status {
  case "ok":
    print("sucesso")
    break
  case "err":
    print("erro")
    break
  default:
    print("desconhecido")
}

Laços

Nova tem várias sintaxes de laço. Todas são expressões da palavra-chave padrão for com diferentes formas.

for i, fim — conta de 0

for i, 5 {
  print(i)   // 0 1 2 3 4
}

for i, inicio, fim — intervalo com direção automática

for i, 1, 6  { print(i) }   // 1 2 3 4 5 6
for i, 5, 1  { print(i) }   // 5 4 3 2 1

for i range(...) — estilo Python

for i range(10)       { }   // 0..9
for i range(2, 8)    { }   // 2..7
for i range(0, 10, 2) { }  // 0 2 4 6 8

for i: passo, inicio, fim — passo personalizado

for i: 2, 0, 10 { print(i) }   // 0 2 4 6 8 10
for i:*2, 1, 32  { print(i) }   // 1 2 4 8 16 32 (multiplicativo)

for...of / for...in

let itens = ["a", "b", "c"]

for item of itens   { print(item) }
for chave  in objeto  { print(chave)  }

Estilo clássico C

for i = 0; i < 10; i++ { print(i) }

while / do...while

while x > 0 { x-- }

do {
  print(x++)
} while x < 5

print é a função de saída embutida do Nova. Ela mapeia para métodos do console com uma sintaxe limpa.

Básico

random()Math.random() → float 0..1
random(max)Float de 0 até max
random(min, max)Float entre min e max
random(array)Elemento aleatório do array
randomInt(max)Inteiro de 0 até max-1
randomInt(min, max)Inteiro entre min e max (inclusive) 
print(random())            // ex.: 0.742
print(random(10))          // ex.: 7.3
print(randomInt(1, 6))    // rolagem de dado: 1..6
print(random(["a", "b", "c"]))  // escolha aleatória

Métodos de Objeto

Objetos comuns têm métodos disponíveis diretamente neles. Todos são não enumeráveis — eles não aparecem em for..in ou Object.keys(). filter, map, find, some, every, e reduce também delegam para os métodos nativos de Array quando chamados em arrays.

.keys()Object.keys(this) — chaves do objeto
.values()Object.values(this) — valores do objeto
.entries()Object.entries(this) — pares [chave, valor]
.has(chave)hasOwnProperty → booleano
.size()Número de chaves próprias
.isEmpty()verdadeiro se não há chaves próprias
.clone()Cópia profunda via structuredClone
.toJson(indent?)JSON.stringify(this, null, indent)
.freeze()Object.freeze(this) — congela o objeto
.merge(...outros)Object.assign({}, this, ...outros) → novo objeto
.assign(...outros)Object.assign(this, ...outros) — modifica o original
.pick(chaves)Novo objeto apenas com as chaves listadas
.omit(chaves)Novo objeto sem as chaves listadas
.invert()Troca chaves por valores (inversão)
.map(fn)fn recebe entrada [chave, valor] — retorna novo objeto
.filter(fn)fn recebe [chave, valor] — retorna objeto filtrado
.find(fn)Primeira entrada onde fn retorna verdadeiro
.some(fn)verdadeiro se alguma entrada satisfaz fn
.every(fn)verdadeiro se todas as entradas satisfazem fn
.reduce(fn, init)Acumula sobre as entradas
.each(fn)Itera sem retornar — retorna this 
let o = { a: 1, b: 2, c: 3 }

print(o.keys())                           // ["a","b","c"]
print(o.has("a"))                         // verdadeiro
print(o.size())                           // 3
print(o.toJson(2))                        // JSON formatado
print(o.merge({ d: 4 }))                  // { a:1, b:2, c:3, d:4 }
print(o.pick(["a","b"]))                  // { a:1, b:2 }
print(o.omit(["c"]))                      // { a:1, b:2 }
print(o.invert())                         // { 1:"a", 2:"b", 3:"c" }

print(o.map(([k,v]) => [k, v * 10]))     // { a:10, b:20, c:30 }
print(o.filter(([k,v]) => v > 1))        // { b:2, c:3 }
print(o.find(([k,v]) => v == 2))         // ["b", 2]
print(o.some(([k,v]) => v > 2))          // verdadeiro
print(o.every(([k,v]) => v > 0))         // verdadeiro
print(o.reduce((acc,[k,v]) => acc+v, 0)) // 6
o.each(([k,v]) => print(k, v))           // itera

dom

O objeto global dom fornece uma API limpa para o DOM do navegador. Disponível apenas no contexto do navegador.

Selecionando elementos

dom.get(seletor)querySelector → NovaElement | null
dom.id(id)getElementById → NovaElement | null
dom.getAll(seletor)querySelectorAll → NovaElementList
dom.classAll(nomeDaClasse)getElementsByClassName → NovaElementList
dom.tag(nomeDaTag)getElementsByTagName → NovaElementList

Criando elementos

// Simples
let btn = dom.create("botao")

// Com atributos
let div = dom.create("div", {
  class: "cartao",
  id: "heroi",
  text: "Olá",
  style: { color: "vermelho" },
  onclick: () => print("clicou")
})

Propriedades do documento

dom.bodydocument.body → NovaElement
dom.headdocument.head → NovaElement
dom.rootdocument.documentElement → NovaElement
dom.titleObter / definir document.title

Eventos & ciclo de vida

// Executa quando o DOM estiver pronto
dom.ready(() => {
  print("DOM carregado")
})

// Evento delegado no documento
dom.on(".btn", "click", (e) => {
  print("botão clicado")
})

// Remover evento
dom.off("click", manipulador)

Utilitários

dom.css(texto)Injeta uma tag <style> no <head>
dom.load(url)Carrega um script → Promise
dom.cookie(nome)Lê um cookie → string | null
dom.cookie(nome, valor, opcoes?)Define um cookie (opções: dias, caminho, seguro)

Métodos de Elemento

Todo elemento retornado por dom.get() é um NovaElement. Os métodos são encadeáveis. Qualquer propriedade nativa do HTMLElement não listada aqui é acessível transparentemente via Proxy.

Classes CSS

let el = dom.get("#caixa")

el.class.add("ativo", "visivel")
el.class.remove("escondido")
el.class.toggle("aberto")
el.class.has("ativo")       // → booleano
el.class.list()               // → ["ativo","visivel"]
el.class("nova-classe")         // substitui todas as classes
print("$el.class")             // lê className como string

Conteúdo

el.text()Lê textContent
el.text("valor")Define textContent — retorna el
el.html()Lê innerHTML
el.html("<b>oi</b>")Define innerHTML — retorna el
el.val()Lê valor do input
el.val("novo")Define valor do input — retorna el

Atributos & Estilo

el.attr("nome")getAttribute → string | null
el.attr("nome", "valor")setAttribute — retorna el
el.id()Lê id
el.id("novo")Define id — retorna el
el.style({color:"vermelho"})Object.assign para el.style
el.style("color:vermelho")Adiciona ao cssText
el.data("chave")Lê dataset.chave
el.data("chave", "valor")Define dataset.chave — retorna el

Eventos

el.on("click", fn)addEventListener — retorna el
el.on("click", fn, opcoes)addEventListener com opções
el.off("click", fn)removeEventListener — retorna el
el.once("click", fn)Dispara uma vez e depois se remove

Árvore DOM

el.append("html")insertAdjacentHTML antes de fechar
el.append(outroEl)appendChild(outroEl._el)
el.prepend(...)insertAdjacentHTML depois de abrir
el.remove()Remove do DOM
el.clone()cloneNode(true) → novo NovaElement
el.find(".sel")querySelector → NovaElement | null
el.findAll(".sel")querySelectorAll → NovaElement[]

Visibilidade & Interação

el.show(display?)style.display = "" (ou valor)
el.hide()style.display = "none"
el.toggle()Alterna entre mostrar e esconder
el.focus()Foca no elemento
el.blur()Desfoca o elemento
el.click()Clique programático
el.scroll(opcoes?)scrollIntoView(opcoes)
el.rect()getBoundingClientRect() → DOMRect
el.elHTMLElement bruto (para bypass)

Exemplo

dom.ready(() => {
  let btn = dom.get("#enviar")
  let msg = dom.get("#mensagem")

  btn.on("click", () => {
    msg
      .text("Enviado!")
      .class.add("sucesso")
    btn.hide()
  })
})

canvas

A função canvas(seletor) envolve um elemento canvas HTML e fornece uma API de desenho totalmente encadeável. 

let c = canvas("#meuCanvas")
  .size(800, 600)
  .fill("#1a1a2e")
  .rect(0, 0, 800, 600)

Estilo

.fill(cor)Define fillStyle (estilo de preenchimento)
.stroke(cor)Define strokeStyle (estilo de traço)
.lineWidth(l)Define a largura da linha
.opacity(v)globalAlpha (0..1) — opacidade global
.font(f)Define a fonte, ex.: "16px sans-serif"
.textAlign(a)"left" | "center" | "right" — alinhamento do texto
.shadow(cor, desfoque, x?, y?)Define sombra
.noShadow()Remove a sombra
.lineCap(c)"butt" | "round" | "square" — estilo da extremidade da linha
.blend(modo)globalCompositeOperation — modo de mesclagem

Formas

.rect(x, y, l, a, r?)Retângulo preenchido (r = raio da borda)
.rectStroke(x, y, l, a, r?)Retângulo com contorno
.circle(x, y, r)Círculo preenchido
.circleStroke(x, y, r)Círculo com contorno
.ellipse(x, y, rx, ry, rot?)Elipse preenchida
.line(x1, y1, x2, y2)Segmento de reta
.lines(x1,y1, x2,y2, ...)Polilinha (pares de coordenadas)
.poly(x1,y1, ...)Polígono preenchido
.polyStroke(x1,y1, ...)Polígono com contorno
.arc(x, y, r, inicio, fim)Arco em graus — com contorno
.arcFill(x, y, r, inicio, fim)Arco — preenchido
.bezier(x1,y1, cp1x,cp1y, cp2x,cp2y, x2,y2)Bezier cúbico
.quadratic(x1,y1, cpx,cpy, x2,y2)Bezier quadrático

Texto

.text(str, x, y, maxL?)fillText — texto preenchido
.textStroke(str, x, y, maxL?)strokeText — texto com contorno
.measureText(str)→ TextMetrics — mede o texto

Transformação

.translate(x, y)Move a origem
.rotate(graus)Rotaciona em graus
.scale(x, y?)Escala (y padrão = x)
.save() / .restore()Empilha / desempilha o estado do canvas
.reset()Reseta a transformação para a identidade

Gradiente

let g = c.linearGradient(0, 0, 800, 0, {
  "0":   "#ff6b6b",
  "0.5": "#feca57",
  "1":   "#48dbfb"
})
c.fill(g).rect(0, 0, 800, 100)

Loop de animação

let c = canvas("#c").size(400, 400)

let parar = c.loop((tempo, quadro) => {
  c.clear("#000")
  c.fill("#fff")
    .circle(200 + Math.sin(tempo / 500) * 100, 200, 20)
})

// stop() cancela a animação
